# Мой прогресс

Я самостоятельно изучаю Linux Administration, готовлюсь к работе в поддержке L1 и постепенно двигаюсь в сторону DevOps.

Мне важно не просто запоминать команды, а понимать причинно следственные связи:

- какой процесс выполняет действие;
- как компоненты взаимодействуют друг с другом;
- почему возникает ошибка;
- как проверить гипотезу;
- как доказать, что проблема действительно устранена.

Для практики я строю собственный production подобный веб сервис FlightSim.
